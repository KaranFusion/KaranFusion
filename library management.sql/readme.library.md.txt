# Library Management System (SQL Project)

## 📌 Overview
This project demonstrates the use of SQL for managing a library database.  
It includes creating tables, inserting data, and performing queries using joins, aggregate functions, and constraints.

## 🛠️ Technologies Used
- MySQL (or any SQL-based database)

## 📂 Database Structure
- **Books** → Stores details about books.
- **Members** → Stores details about library members.
- **IssuedBooks** → Tracks issued books and their return status.

## 🚀 How to Run
1. Open MySQL Workbench or any SQL tool.
2. Copy and paste the contents of `library_management.sql`.
3. Execute the script.
4. Run the sample queries to view results.

## 📊 Sample Queries
- List all books
- Show all issued books with member names
- Count total books
- Find overdue books
